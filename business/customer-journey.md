 # Customer-Journey.md

## Aware

### Trigger Event
- Game developers encounter a project with tight deadlines or limited resources, requiring a quick solution for temporary or intermediate game development.
- Game developers face challenges in integrating AI tools seamlessly into their existing workflow.

### Friction Points
- Lack of awareness about the existence of a tool specifically designed for game development with AI.
- Skepticism about the effectiveness of AI in game development.

### User Emotions
- Frustration due to time constraints and resource limitations.
- Anxiety about the potential impact of integrating new tools on their existing workflow.

### Opportunities to Delight
- Clear and concise messaging about the benefits of using game-scaffold for quick, efficient, and AI-assisted game development.
- Demonstrating the tool's compatibility with existing workflows and its minimal impact on final product integrity.

### Metric
- Impressions on targeted game development forums and social media platforms.

## Consider

### Trigger Event
- Game developers research potential solutions for their temporary or intermediate game development needs.

### Friction Points
- Comparison of game-scaffold with other AI tools or traditional methods.
- Concerns about the learning curve associated with a new tool.

### User Emotions
- Curiosity about the capabilities of game-scaffold.
- Caution about the potential complexity of learning a new tool.

### Opportunities to Delight
- Providing clear and easy-to-understand documentation and tutorials.
- Offering a free trial or demo to showcase the tool's capabilities.

### Metric
- Engagement with tutorials, demos, and documentation.

## Try

### Trigger Event
- Game developers decide to try game-scaffold for their project.

### Friction Points
- Learning curve associated with using a new tool.
- Integration issues with existing game development tools or workflows.

### User Emotions
- Excitement about the potential benefits of using game-scaffold.
- Frustration due to learning curve or integration issues.

### Opportunities to Delight
- Providing prompt and helpful customer support.
- Offering integration assistance to ensure seamless compatibility with existing tools and workflows.

### Metric
- Number of trial downloads or usage.

## Adopt

### Friction Points

### User Emotions

- Gratitude for the time and resource savings.

### Opportunities to Delight
- Continued support and updates to ensure the tool remains effective and relevant.
- Encouraging user feedback and incorporating suggestions for improvements.

### Metric
- Number of active users or regular downloads.

## Expand

### Friction Points

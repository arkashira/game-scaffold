# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User (CPU)
- **Compute Costs**: $0.10 per hour per user
  - Average usage: 10 hours/month
  - Monthly compute cost per user: $0.10 * 10 = **$1.00**
  
- **Storage Costs**: $0.02 per GB per month
  - Average storage per user: 5 GB
  - Monthly storage cost per user: $0.02 * 5 = **$0.10**
  
- **Bandwidth Costs**: $0.01 per GB
  - Average bandwidth usage: 20 GB/month
  - Monthly bandwidth cost per user: $0.01 * 20 = **$0.20**

**Total Cost per Active User (CPU)**:  
$1.00 (compute) + $0.10 (storage) + $0.20 (bandwidth) = **$1.30**

---

### Pricing Tiers
| Tier Name        | Price ($/mo) | Features                                                                 |
|------------------|---------------|--------------------------------------------------------------------------|
| **Basic**        | $9.99        | Access to AI tools, Community support, 10 GB storage, 10 hours compute  |
| **Pro**          | $29.99       | All Basic features, 50 GB storage, 50 hours compute, Priority support   |
| **Enterprise**    | $99.99       | All Pro features, 200 GB storage, Unlimited compute, Dedicated support   |

---

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user
  - Marketing spend, sales efforts, and promotional activities considered.

---

### Lifetime Value (LTV) Estimate
- Average subscription length: 24 months
- Average Monthly Revenue per user (ARPU):
  - Basic: $9.99
  - Pro: $29.99
  - Enterprise: $99.99
- LTV Calculation:
  - Basic: $9.99 * 24 = **$239.76**
  - Pro: $29.99 * 24 = **$719.76**
  - Enterprise: $99.99 * 24 = **$2399.76**

---

### Break-even Users Count
- Break-even point = CAC / (ARPU - CPU)
  
For each tier:
- **Basic**:  
  Break-even users = $50 / ($9.99 - $1.30) = **6.55** ~ **7 users**  
  Break-even users = $100 / ($9.99 - $1.30) = **13.10** ~ **14 users**

- **Pro**:  
  Break-even users = $50 / ($29.99 - $1.30) = **1.83** ~ **2 users**  
  Break-even users = $100 / ($29.99 - $1.30) = **3.66** ~ **4 users**

- **Enterprise**:  
  Break-even users = $50 / ($99.99 - $1.30) = **0.50** ~ **1 user**  
  Break-even users = $100 / ($99.99 - $1.30) = **1.01** ~ **2 users**

---

### Path to $10K MRR
To achieve $10,000 MRR, we can consider the following scenarios:

- **Basic Tier**:  
  $10,000 / $9.99 = **1001 users**

- **Pro Tier**:  
  $10,000 / $29.99 = **334 users**

- **Enterprise Tier**:  
  $10,000 / $99.99 = **100 users**

**Recommendation**: Focus on the **Pro Tier** to balance user acquisition and revenue generation, targeting **334 users** for sustainable growth.
